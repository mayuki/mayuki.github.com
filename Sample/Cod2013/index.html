Internet Explorer 10 と Windows ストアアプリ - Community Open Day 2013 Samples

Samples

サンプルは基本的にInternet Explorer 10でのみ動作します(あくまでサンプルだからです)。

Grid
Grid Layoutの例です。
Pointer-Events
マウスやタッチ、ペンでの操作を取得します。
Pointer-Events-Pressure
ペンを利用したときの筆圧情報を取得して表示・反映します。
CSS Transition
画像にマウスをホバーするとプロパティの値が変わってCSS Transitionが発動します。
WebWorkers
画像をセピア調にする処理を行い、UIをブロックしてしまう有無を確認できます。
CSS Animation
ボタンをクリックしたり、画像にマウスをホバーするとCSS Animationが発動します。
Drag-and-Drop
画像ファイルをブラウザにドロップすると画像データを読み込んで表示します。
FileAPI
画像ファイルを選択するとJavaScriptで画像データを読み込んで表示します。
Gesture-Events
マルチタッチジェスチャー(ピンチ、ホールド、回転)を試すことができます。

Application

サンプルを合体したアプリ
画像をドロップして貼り付け、マルチタッチジェスチャーで画像を移動し、ペンで筆圧を感知しつつ線を書くことができ、セピア調に変換しつつ、画像をダウンロードできるというものです。
